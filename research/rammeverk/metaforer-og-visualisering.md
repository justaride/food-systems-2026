# Metaphors & Visualization Concepts for Food Systems

**Date:** December 30, 2025
**Purpose:** Brainstorm new ways to understand and visualize the complete Norwegian food supply chain

---

## Part I: Metaphor Exploration

                                      │
                    ┌─────────────────┴─────────────────┐
                    ▼                                   ▼
            ┌───────────────┐                 ┌───────────────┐
            │   ARTERIES    │                 │   ARTERIES    │
            │   E6 North    │                 │   E6 South    │
            │   E18 West    │                 │   E39 Coast   │
            └───────────────┘                 └───────────────┘
                    │                                   │
        ┌───────────┴───────────┐           ┌──────────┴──────────┐
        ▼           ▼           ▼           ▼          ▼          ▼
    ┌───────┐   ┌───────┐   ┌───────┐   ┌───────┐  ┌───────┐  ┌───────┐
    │Regional│   │Regional│   │Regional│   │Regional│  │Regional│  │Regional│
    │ Hubs  │   │ Hubs  │   │ Hubs  │   │ Hubs  │  │ Hubs  │  │ Hubs  │
    └───────┘   └───────┘   └───────┘   └───────┘  └───────┘  └───────┘

    │                    Last-mile to consumers                        │

**What flows:**
- Blood = Food products
- Oxygen = Value/money

- Atherosclerosis = Infrastructure bottlenecks
- Heart attack = Hub failure
- Stroke = Regional supply cut
- Anemia = Insufficient variety

**Visualization:** Animated flow showing "blood" moving from heart to extremities, with color indicating freshness/time.

### 1.2 River Systems

**Core Metaphor:** Food flows like water in a watershed

```
                          TRIBUTARIES
                    ┌─────────────────────────────┐
                    │   Farms    Fisheries        │
                    │      \        /             │
                    │       \      /              │
                    │        \    /               │
                    │    Processors               │
                    │         |                   │
                    │         ▼                   │
                    └─────────────────────────────┘
                              │
                              ▼
                    ┌─────────────────────────────┐
                    │       MAIN RIVER            │
                    │   Logistics Networks        │
                    │   (ASKO, Coop, Rema)        │
                    └─────────────────────────────┘
                              │
                    ┌─────────┴─────────┐
                    ▼                   ▼
            ┌───────────────┐   ┌───────────────┐
            │    DELTA      │   │    DELTA      │
            │  Urban Areas  │   │  Rural Areas  │
            │ Dense stores  │   │ Sparse stores │
            └───────────────┘   └───────────────┘
                    │                   │
                    ▼                   ▼
            ┌───────────────────────────────────┐
            │              OCEAN                 │
            │           5.4M consumers           │
            └───────────────────────────────────┘
```

**Key Concepts:**
- Watershed = Regional supply catchment
- Tributaries = Multiple producer streams
- Confluence = Where streams merge (processing)
- Main river = Primary logistics channel
- Dam = Wholesale bottleneck
- Delta = Consumer distribution
- Flood plains = Surplus handling

**Visualization:** Map overlay showing "water flow" from production regions to consumption centers.

### 1.4 Geological Processes

**Core Metaphor:** Market dynamics as tectonic forces

```
    ┌─────────────────────────────────────────────────┐
    │                    CRUST                         │
    │   Visible retail layer (stores, prices)          │
    └─────────────────────────────────────────────────┘
    ┌─────────────────────────────────────────────────┐
    │                   MANTLE                         │
    │   Logistics, wholesale, infrastructure           │
    └─────────────────────────────────────────────────┘
    ┌─────────────────────────────────────────────────┐
    │                    CORE                          │
    │   Ownership, capital, political connections      │
    └─────────────────────────────────────────────────┘

         NorgesGruppen      Coop        Rema
         ◄──────────►   ◄──────────►   ◄──────────►
           PLATE          PLATE          PLATE
           43%            30%            25%
```

**Concepts:**
- Tectonic plates = Company territories (market share)
- Fault lines = Competitive boundaries
- Earthquakes = Disruptive events (new entrants, scandals)
- Volcanic activity = Innovation/disruption
- Erosion = Gradual market share shifts
- Sedimentation = Infrastructure accumulation

**Visualization:** Plate boundaries showing where companies compete, with historical animation of boundary shifts.

---

### 1.5 Weather Systems

**Core Metaphor:** Food flows like atmospheric dynamics

**Concepts:**
- High pressure = Production surplus areas
- Low pressure = Consumption centers (demand pull)
- Wind = Transport flows from high to low pressure
- Fronts = Seasonal transitions
- Storms = Disruption events
- Drought = Supply shortage

**Visualization:** Weather map style showing "pressure systems" and "wind patterns" of food movement.

---

### 1.6 Forest Ecosystem

**Core Metaphor:** The food system as forest with visible and invisible layers

**Layers:**
1. **Canopy** (visible) = Retail storefronts
2. **Understory** = Regional logistics
3. **Forest floor** = Producers
4. **Root system** = Infrastructure, capital
5. **Mycorrhiza** (invisible) = Hidden connections, shared systems

**Dynamics:**
- Light competition = Shelf space competition
- Nutrient cycling = Value circulation
- Decomposition = Food waste processing
- Succession = Market evolution

## Part II: Visualization Concepts

### 2.1 Animated Flow Map

**Concept:** Real-time particle flow showing food movement

```
┌─────────────────────────────────────────────────────────────┐
│                        NORWAY MAP                            │
│                                                              │
│   ● Farms (green particles emit from here)                  │
│   ● Ports (blue particles enter from abroad)                │
│   ● Hubs (particles converge, transform, redistribute)      │
│   ● Stores (particles terminate here)                       │
│                                                              │
│   Particle color:                                            │
│   🟢 Fresh produce    🔵 Seafood    🔴 Meat                  │
│   🟡 Dairy           🟤 Grains     ⚪ Imports               │
│                                                              │
│   Animation: Particles flow along actual road network       │
│   Speed: Represents time-to-market                          │
│   Density: Represents volume                                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**Technical Approach:**
- Use deck.gl TripsLayer for animated particles
- Road network geometry from OSM
- Particle emission rate proportional to production
- Color gradient for freshness decay
